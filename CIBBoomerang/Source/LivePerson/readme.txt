Content of this example

chatApp4Model - implements chat communications
chatApp4Message - represents one chat line
ErrorXMLParser - parse error XML messages
EventsXMLParser - parse the events XML and builds array of chatApp4Message
InfoXMLParser - parse the Info XML


modelUsageExample - Usage example of the model. This example will not compile. It does not have the implementation of most of the 
UI elements. The important thing about this example is the usage of a timer to update the UI and the way to call and initialize the model.
